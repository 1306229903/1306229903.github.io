canvas绘图在高清屏出现模糊的问题

2021-05-06 1 min read # html+css

canvas 标签中的 width 和 height 属性并不是 css 中的宽高，而是 canvas 绘图上下文（绘图区域）的宽高。
CSS宽高会决定所占物理区域的宽高，画布分为绘图区域和实际页面所占物理区域，绘图的时候会将绘制的图形根据一定缩放比率进行缩放，它会将整个 canvas 绘图区域塞进 canvas css 宽高中并填满，这样有时就会出现模糊现象。
解决方案为：首先分别声明 canvas 的 css 宽高和上下文宽高，同时上下文宽高应该是 css 宽高的 devicePixelRatio （屏幕设备比）的倍数
